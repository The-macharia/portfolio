Fashion MNIST Neural Network

This program was created using python for machine learning and the Tensorflow algorithm. It uses the FASHION MNIST dataset consisting of 60,000 images 28 x 28 greyscale images. 50,000 for the training set and 10,000 for the testing set. It can predict the correct label of clothing articles with an accuracy of 96%. The clothing articles are divided into 10 groups; T-shirt/top, Trouser, Pullover, Dress, Coat, Sandal, Shirt, Sneaker, Bag, Ankle boot.

CIFAR-10 IMAGE CLASSIFIER

This is an image classifier also built using python for machine learning and the Tensorflow algorithm and can classify an image with an accuracy of 98%. The CIFAR-10 dataset contains 60,000 32x32 color images in 10 different classes. The 10 different classes represent airplanes, cars, birds, cats, deer,dogs, frogs, horses, ships, and trucks. There are 6,000 images of each class. The CIFAR-10 dataset is a collection of images that are commonly used to train machine learning and computer vision algorithms.
